# Guia de Download de Animações Lottie

## Ícones Necessários

Baixe as seguintes animações do LottieFiles.com e salve na pasta `assets/lottie/`:

### Navegação (Navigation Bar)

1. **Home Icon** - `home.json`
   - URL: https://lottiefiles.com/search?q=home%20icon&category=animations
   - Procure por: "Home icon animation" (escolha um com efeito de pulse suave)

2. **Credit Card** - `credit_card.json`
   - URL: https://lottiefiles.com/search?q=credit%20card&category=animations
   - Procure por: "Credit card animation" (escolha um com shimmer effect)

3. **QR Scanner** - `qr_scan.json`
   - URL: https://lottiefiles.com/search?q=qr%20scanner&category=animations
   - Procure por: "QR code scanner" (escolha um com linhas de scan)

4. **Analytics/Chart** - `analytics.json`
   - URL: https://lottiefiles.com/search?q=pie%20chart&category=animations
   - Procure por: "Pie chart animation" (escolha um com rotação suave)

5. **Profile/User** - `profile.json`
   - URL: https://lottiefiles.com/search?q=user%20profile&category=animations
   - Procure por: "User profile icon" (escolha um com pulse effect)

### Analytics Stats

6. **Bitcoin** - `bitcoin.json`
   - URL: https://lottiefiles.com/search?q=bitcoin&category=animations
   - Procure por: "Bitcoin icon animation"

7. **Trending Up** - `trending_up.json`
   - URL: https://lottiefiles.com/search?q=trending%20up&category=animations
   - Procure por: "Arrow up trend"

8. **Bar Chart** - `bar_chart.json`
   - URL: https://lottiefiles.com/search?q=bar%20chart&category=animations
   - Procure por: "Bar chart animation"

### Card Actions (My Cards Screen)

9. **Freeze/Snowflake** - `freeze.json`
   - URL: https://lottiefiles.com/search?q=snowflake&category=animations

10. **Lock** - `lock.json`
    - URL: https://lottiefiles.com/search?q=lock&category=animations

11. **Speed/Gauge** - `speed.json`
    - URL: https://lottiefiles.com/search?q=speedometer&category=animations

12. **Settings** - `settings.json`
    - URL: https://lottiefiles.com/search?q=settings%20gear&category=animations

## Como Baixar

1. Acesse o link de cada ícone
2. Clique no ícone que você gostar
3. Clique em "Download" ou "Free Download"
4. Escolha o formato "Lottie JSON"
5. Salve o arquivo com o nome indicado acima em `assets/lottie/`

## Alternativa Rápida

Se preferir, você pode usar ícones estáticos por enquanto. O código já tem fallback automático para Material Icons caso os arquivos Lottie não existam.
